# Road Traffic Conditions Detection System

A complete full-stack web application for detecting road traffic conditions using audio analysis and machine learning.

## Features

- 🎵 **Audio Analysis**: Extract MFCC, ZCR, and spectral features using librosa
- 🤖 **Machine Learning**: Decision Tree model for accurate traffic prediction
- 🔍 **Location Search**: Autocomplete search with 5000+ locations
- 🗺️ **Route Analysis**: Source to destination traffic prediction
- 📊 **History Tracking**: Store and visualize search history
- 🔔 **Smart Alerts**: Real-time alerts for heavy traffic
- ⚙️ **Customizable Settings**: Adjust refresh intervals and alerts
- 📱 **Responsive Design**: Works on all devices

## Tech Stack

### Frontend
- React 18 (Functional Components + Hooks)
- React Router DOM
- Axios
- Chart.js
- Modern CSS

### Backend
- Python Flask
- SQLite Database
- Librosa (Audio Analysis)
- scikit-learn (Decision Tree)
